# 🧠 AI-Powered Product Data Optimizer (Mini PIM System)

## 🚀 Overview
This project is an AI-powered Product Information Management (PIM) system that improves incomplete or low-quality product data for e-commerce platforms.

It analyzes product inputs, detects issues, enhances descriptions, generates missing attributes, and evaluates overall data quality using AI.

---

## 🎯 Problem Statement
E-commerce platforms often suffer from incomplete or inconsistent product data, which leads to:
- Poor search visibility
- Low conversion rates
- Inefficient catalog management

This project solves that by automatically enhancing product data using AI.

---

## 💡 Solution
The system takes raw product input and performs:

1. 🔍 Issue Detection  
   Identifies missing or weak fields (category, description, attributes)

2. 🤖 AI Enhancement  
   Improves product name, description, and generates relevant attributes

3. 🧠 AI Evaluation (Agentic AI)  
   Assigns a quality score and provides feedback

4. 📊 Rule-Based Scoring  
   Measures completeness of product data

5. 📦 Data Storage (PIM Feature)  
   Stores original and improved product data in a database

6. 🔗 WooCommerce Integration (Simulated)  
   Demonstrates syncing product data to e-commerce platforms

---

## 🧠 Key Features

- AI-based product data improvement
- Dynamic attribute generation (no hardcoding)
- Agentic AI evaluation system
- SQLite-based product storage
- Streamlit interactive UI
- WooCommerce API simulation

---

## ⚙️ Tech Stack

- Python
- Streamlit (UI)
- SQLite (Database)
- Groq API (LLM - LLaMA models)

---

## 🧩 System Architecture

```text
User Input
   ↓
Issue Detection
   ↓
AI Enhancement
   ↓
AI Evaluation
   ↓
Store in Database
   ↓
Display + WooCommerce Sync

```
